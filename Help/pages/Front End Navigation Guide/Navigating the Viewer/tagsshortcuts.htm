Tags & Shortcuts


  1. Info Tags: Most of the objects on the glyph are “tagged” with data. Info tags are displayed as text labels on screen that allow you to see the underlying source data mapped to a specific glyph object. It’s useful to use tags when you’re interested in looking at one or two specific glyph object details.

    • Turning on and off into tags:

      • On: click "i" on the keyboard
      • Off: click "o" on the keyboard

  2. Refreshing Visualization: F5
  3. Closing a Visualization: CTRL+F4
  4. Getting back to the Selection menu: from the visualization screen, click file -> close visualization OR CTRL+F4
