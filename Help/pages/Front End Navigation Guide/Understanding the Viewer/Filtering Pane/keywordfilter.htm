Keyword Filtering

    • Click Add -> choose field -> enter text to filter on.
    • Can use text filters mutiple times per field.
    • Can add multiple keyword filters.
    • Individual filters in the keyword pane can be removed by right clicking the filter -> remove.
    • Click Remove All Filters to remove all of your selections.
    • Click on Update 3D View to refresh the view to show only glyphs that match your filters or click Show Filtered Data, Create Subset Visualization, or Show Selected Data (see below for description of these options).
