Elastic Filtering

    • Scroll through Field Values list -> choose field -> click Show All to display all values -> click as many options you like. Hold CTRL + select (single non-contiguous) or SHIFT + select (contiguous entries), or click the Show All box to select all values.
    • This can be repeated for multiple fields.
    • If you want to clear all selections, click on the Clear All FIlters button at the top of the Filtering toolbar.
    • Click Show Filtered Data, Create Subset Visualization, or Show Selected Data (see below for description of these options).
